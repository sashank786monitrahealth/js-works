Advantage of a Module

  • Avoid Collisions (Namespace)
  • Reusability - because a module is self-contained
  • Maintainability - A module is also easier to maintain. This is because a module is decooupled from rest of the code.

Example:

A function is fantastic structure for module.

Anything inside the function will have access to variables etc. Even when the function is done, that is, no longer executing.This is because of closure.

We still have access to the variables and the functions - because of closure. Those two concepts make the structure of a function perfect for setting up a module - which is a set of self contained code.

Many times when module pattern is used in JavaScript. It is used as a singleton. It is used only once.
