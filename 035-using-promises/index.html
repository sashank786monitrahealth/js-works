What is a promise

A promise is simply a JavaScript object:

  • An object with properties and methods
  • We use some of those methods to take advantage of promises.
  • Represents the eventual completion (or Failure) of an Aynchronous Operation.
  • Promise will always provide a value - When the completion occurs, when the future event occurs - it provides a value.
